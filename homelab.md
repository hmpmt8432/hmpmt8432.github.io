# Homelab / Cybersecurity Projects

This section documents the buildout of my cybersecurity homelab and related security projects. The goal of the lab is to simulate a small enterprise-style environment where I can practice implementation, monitoring, hardening, troubleshooting, and documentation.

Rather than treating the homelab as a collection of random tools, I am approaching it as a structured program with multiple related projects. Each project builds on the previous one and contributes to a more complete environment for security learning and operational practice.

## Homelab Goals

The homelab is being built to support learning and demonstration in the following areas:

- network architecture and segmentation
- firewall deployment and policy control
- Windows Server administration
- Active Directory
- domain-joined endpoint management
- centralized logging and security visibility
- vulnerability management
- incident response workflows
- documentation and project tracking
